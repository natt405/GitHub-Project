Github Managment

How can I manage projects with Github? October 31st 2019

Project management in Github is made simple due to several components that are integrated into the web page itself. Github has its own project management section that they provide. The application allows you to plan your project, sort tasks, automate your workflow, track your progress, share your status, and wrap up your project.The project stlye used is a Kanban Board.


How to use Project Managment

When creating your project you will name it, describe it, and then choose a project template that fits your needs or style. After creating this project which Github calls a Project Board you can then see the columns that are in it which are the “To do”, the “Progress”, and the “Done” columns. Using these columns to view your work flow is really easy and you can also add notes and issues on them when they are needed. In addition you can also setup automations so you can automatically moves issues through their lifecycle without having to perform the operations manually.
